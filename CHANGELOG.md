# 更新日志

YMOS（勇麦投资操作系统）的所有重要变更均记录于此。

## [未发布]

### 新增功能
- **价格监控功能** — 通过富途 OpenD 实现自动市场盯盘
  - P2 阶段转换防护，包含验证检查
  - P6 止损建议，配备安全护栏
  - 扩展行情支持，自动识别交易时段
  - 市场-板块-个股三层信号联动
  - 自动盯盘工作流设计规范

- **Skills 架构 V4** — Claude Code 原生发现机制
  - 将所有 skills 从 `.agents/skills/` 迁移至 `.claude/skills/`
  - Skills 现可通过 `/ymos-*` 斜杠命令发现
  - 清理了遗留的 V3 路径和目录

- **富途 OpenD 集成** — 扩展市场数据能力
  - 3 个新能力：情绪分析、资金流向追踪、股票筛选器
  - 4 个富途 OpenD 操作的 CLI 命令
  - 修复了 8 个 skills CLI 命令错误
  - 更新了入门引导、README 和指南文档

- **技术分析基础** — 基于图表的分析（已延期）
  - 技术指标和信号框架
  - 多周期报告生成
  - 功能设计规范（已归档，0/20 任务完成）

- **FinceptTerminal 改进** — 吸纳 5 项优化
  - 数据抓取优化
  - 报告生成改进

### 改进
- **Token 优化** — 将大型 MD 文件拆分为索引 + 子文件
  - 减少 skill 文档的 token 消耗
  - 提升复杂工作流的加载速度

### 修复
- 修复了 6 个 skill 文件中过时的 V3 路径引用
- 将所有 V3 路径引用（Eyes/Brain/持仓与关注）迁移至 V4 架构

### 移除
- 移除了遗留的富途 skill（已被集成的富途 OpenD 能力替代）
