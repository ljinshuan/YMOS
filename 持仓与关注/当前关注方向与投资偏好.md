## 三、双轨并行策略

### 轨道一：长线 PVE（主轴）
- 盯着值得长期持有的各赛道龙头
- 等大跌 / 错杀机会才建仓，不追高
- 进入标准：基本面扎实 + 价格有显著折价
- **持有标准**：买入逻辑未被证伪 + 基本面趋势未恶化

### 轨道二：波段 PVP（辅助，严格纪律约束）
- 只做有**板块 Beta 向上**支撑的板块龙头
- 先看板块势能（市场洞察 → 投资雷达），再看个股位置
- 板块势能消失则立即退出，不硬持
- **绝对前置条件**：板块热度至少连续 2 日出现在洞察报告中
- **退出标准**：板块热度从洞察报告消失 或 个股跌破关键支撑

- **入池条件**：美股热门股票 + 从热门板块逆向分析出的上下游潜力标的
- **出池条件**：财报暴雷 或 行业下行趋势确认

- **核心策略**：不追热门本身，而是挖掘热门板块上下游尚未被关注的标的

## 七、策略执行偏好

- 买入/加仓前强制前置：**板块热度确认 + P2 + 个股知识库 + 当前关注方向与投资偏好**
- 卖出/减仓前强制前置：**行业下行确认 + P2 + 读取买入理由匹配 + P12 纪律审查**
- 纪律强制：任何动作需 P12 审查
- **自动建议触发阈值**：个股单日异动 ≥ ±5%，或板块连续 2 日在洞察中出现
- **特别关注**：用户恋战/死扛倾向，系统在整体市场走弱时应主动提醒减仓/空仓选项

## 八、当前禁忌

- **不追涨、不在高估值区间建仓**（宁可踏空，不追顶）
- **不恋战**：拿着不舒服的标的，第一时间退出
- **不接飞刀**：除非极度超跌且基本面未恶化
- 其他禁忌：暂无，后续积累经验后补充

## 九、核心心法

> 从历史交易中提炼的核心投资信念，供 AI 在策略建议时作为"灵魂校准"参考。

### 关键教训
- **死扛是最贵的错误**：整体走弱时不愿空仓，导致亏损扩大。系统应在市场转弱时优先提醒减仓/空仓选项。
- **恋战要命**：亏损时不愿止损是最大弱点，纪律必须严格执行。

### 投资信念
- 从热门板块逆向分析上下游，在市场尚未关注时提前布局
- 波段操作为主，不贪长线的故事
