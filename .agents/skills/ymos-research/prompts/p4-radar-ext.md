### 3.2 竞品信号解读模板

> **每次竞品发布财报或重大消息时，用以下框架快速解读对 [{{ticker}}] 的影响：**

```
竞品事件：[XX 公司发布了什么]
├── 行业验证？ → 这是否验证了整个行业的需求强度？（利好行业 = 利好目标公司）
├── 份额威胁？ → 竞品的增长是否以目标公司的份额为代价？（份额流失 = 利空）
├── 定价信号？ → 竞品是在提价还是降价？（降价 = 利润率压力传导）
└── 技术路线？ → 竞品是否验证了不同的技术路线？（技术分叉 = 需重新评估护城河）
```

|---------|---------|-------|---------|
| 利率变化 | [对融资成本/折现率/NII 的影响] | 高/中/低 | Fed 会议纪要、利率期货 |
| 汇率波动 | [对海外收入/成本的影响] | 高/中/低 | 外汇市场 |

### 5.2 行业宏观先行指标

> **根据目标公司所在行业，从下表中选取对应的宏观先行指标加入雷达。这些指标往往比公司自身的财务数据更早反映需求/成本的变化方向。**

| 行业 | 核心先行指标 | 数据来源 | 对公司的传导逻辑 |
|------|------------|---------|----------------|
| **制造业/工业** | ISM Manufacturing PMI（荣枯线 50）、新订单分项、库存分项 | ISM 月度报告 | PMI >50 且新订单扩张 → 工业需求回暖 → 订单增加；库存分项下降 → 补库需求 |
| | 工业产出指数（Industrial Production） | Fed 月度报告 | 产出环比变化 → 制造业景气度的硬数据验证 |
| | 耐用品订单（Durable Goods Orders） | Census Bureau 月度 | 核心资本品订单（ex-defense, ex-aircraft）= 企业 CapEx 意愿的先行指标 |
| **消费/零售** | 消费者信心指数（Univ. of Michigan / Conference Board） | 月度 | 信心下降 → 可选消费支出减速 → 零售/消费品公司收入承压 |
| | 非农就业 + 时薪增速 | BLS 月度 | 就业强 + 薪资涨 → 消费能力支撑；失业率升 → 消费收缩 |
| | 零售销售数据（Retail Sales） | Census Bureau 月度 | 实际消费支出的直接度量，区分 auto/ex-auto/core |
| | 个人消费支出（PCE） | BEA 月度 | Fed 偏好的通胀/消费指标，实际 PCE 增速 = 消费动能 |
| **建筑/地产** | 新屋开工（Housing Starts）+ 营建许可（Building Permits） | Census Bureau 月度 | 许可 = 最先行信号 → 开工 = 确认信号 → 建材/设备需求传导 |
| | 新屋销售 + 成屋销售 + 待完成销售（Pending Home Sales） | Census/NAR 月度 | 销售量 → 下游装修/家居/家电需求传导 |
| | NAHB 住宅市场指数（Builder Confidence） | NAHB 月度 | 建筑商信心 = 未来开工意愿的情绪先行指标 |
| | 30 年期按揭利率 | Freddie Mac 周度 | 利率 ↑ → 购房负担加重 → 需求抑制 → 开工减速 |
| **航空航天/国防** | 全球航空旅客量（RPK）/ 航空货运量（FTK） | IATA 月度 | RPK 恢复至 2019 水平以上 → OEM 产能爬坡信心增强 → 材料/零部件订单增加 |
| | 波音/空客交付量 + 积压订单（Backlog） | OEM 月度公布 | 交付加速 = 供应链拉动；积压增长 = 长期需求可见度 |
| | 美国国防预算 / NATO 国防支出占 GDP 比例 | 政府年度预算、NATO 公告 | 国防预算增长 → 军工订单增加 → 特种材料/系统需求 |
| **科技/半导体** | 全球半导体销售额（WSTS） | SIA/WSTS 月度 | 行业收入增速 = 半导体周期位置的直接度量 |
| | 半导体设备出货额（SEMI BBR） | SEMI 月度 | BB ratio >1 → 行业在扩张产能 → 上游设备/材料需求强劲 |
| | Cloud CapEx（AWS/Azure/GCP 季度资本开支） | Hyperscaler 季报 | Cloud CapEx 加速 → 服务器/芯片/光模块需求增加 |
| **能源** | WTI / Brent 原油价格 | 实时 | 油价直接影响上游 E&P 公司收入；影响下游炼化利润率 |
| | 天然气价格（Henry Hub / TTF） | 实时 | 区域价差反映供需平衡和地缘风险 |
| | 美国活跃钻机数（Baker Hughes Rig Count） | Baker Hughes 周度 | 钻机数 = 上游活动的先行指标 → 油服公司收入前瞻 |
| | OPEC+ 产量决策 + 美国原油库存（EIA Weekly） | OPEC 会议 / EIA 周度 | 减产 → 供给收紧 → 油价支撑；库存超预期累积 → 需求疑虑 |
| **金融/银行** | 收益率曲线形态（2Y-10Y Spread） | 实时 | 曲线倒挂 → 银行 NIM 承压 + 衰退信号；变陡 → NIM 扩张 |
| | 贷款需求（Fed Senior Loan Officer Survey） | 季度 | 贷款标准收紧 → 信贷收缩 → 经济放缓先行信号 |
| | 信用利差（IG/HY Spread） | 实时 | 利差走阔 → 风险偏好下降 → 信贷环境收紧 |
| **Fintech/零售经纪** | 零售交易量指数（CBOE equity volume、options volume） | 交易所每日 | 散户交易活跃度 = 交易收入的直接先行指标 |
| | BTC / 总 crypto 市值 | 实时 | Crypto 牛市 → crypto 交易收入增加；熊市 → 反之 |
| | VIX（恐慌指数） | 实时 | 适度波动（VIX 15-25）→ 交易活跃；极端高（>35）→ 避险情绪压制开户/交易 |
| **医药/生物科技** | FDA 审批日历（PDUFA dates） | FDA 公布 | 关键药物审批通过/拒绝 = 该公司及竞品估值的重大催化剂 |
| | 医保政策（CMS 定价/报销政策） | CMS 年度更新 | 药品定价压力 → 收入预期调整 |
| **物流/运输** | BDI（波罗的海干散货指数） | 实时 | 全球贸易活跃度和散货运价的晴雨表 |
| | 集运运价指数（SCFI / Freightos FBX） | 周度 | 运价 = 集运公司利润的直接驱动力 |
| | 美国卡车运量指数（ATA Truck Tonnage） | ATA 月度 | 国内货运需求 = 经济活动的同步指标 |
| **电力/公用事业** | 电力需求增速（EIA 月度数据） | EIA 月度 | 数据中心 + 电气化 → 电力需求结构性增长信号 |
| | 电网互联审批（PJM / ERCOT 队列） | 区域电网运营商 | 新增电网建设审批 → 电力设备/输配电需求增加 |
| | 天然气价格 → 电价传导 | 实时 | 天然气占边际发电成本 → 气价涨推高电价 → 公用事业收入 |

**使用说明**：
- 生成某个标的的雷达时，从上表中选取**该行业对应的 2–4 个最相关的先行指标**，填入 5.1 的宏观敏感因素表中
- 如果公司跨多个行业（如 Robinhood 跨 Fintech + 消费），从多个行业中各选 1–2 个最相关的
- 每个指标都必须标注**对该公司的具体传导逻辑**，而不是泛泛地说"利率影响大"
