# P22: 衍生品异动信号分析 (Derivatives Anomaly)

## 路径上下文（YMOS）
- 根目录：`YMOS/`
- 输入：`ymos fetch-derivatives-anomaly` 输出的 JSON 数据
- 可选输入：标的 `个股基础知识库.md`（提供基本面上下文）
- 写回：投资雷达报告「衍生品信号」区块 + 状态机 P4 更新

## 适用场景

在投资雷达的技术面扫描步骤之后，对持仓和 Watchlist 标的衍生品异动数据进行分析。仅适用于港股（牛熊证 + 期权）和其他市场的期权异动。

## 输入数据来源

由 `ymos fetch-derivatives-anomaly fetch --from-state` 生成，包含：
- **牛熊证异动**（仅港股）：街货比例变化、价格区间异动
- **期权异动**：五维信号（大单/波动率/量价/情绪/综合）

# Analysis Protocol

按以下维度分析，仅在有明确信号时报告：

## 维度一：期权异动（五维信号）

### 1.1 大单信号（option_unusual）
**检测规则：**
- 单笔期权交易量超过日均量 3 倍 → 大单异动
- 看涨期权大单买入 → 机构看多
- 看跌期权大单买入 → 机构看空/对冲

**强度评级：**
- **strong**：大单量 > 日均 5 倍 + 方向集中
- **moderate**：大单量 > 日均 3 倍
- **weak**：偶发大单，方向分散

### 1.2 波动率信号（option_volatility）
**检测规则：**
- 隐含波动率（IV）突升 > 30% 日环比 → 市场预期剧变
- IV 与历史波动率（HV）显著背离 → 定价偏差
- IV skew 异常（看涨/看跌 IV 差值扩大）→ 方向性预期

### 1.3 量价信号（option_volume_price）
**检测规则：**
- 看涨期权成交量激增 + 正股上涨 → 多头共识
- 看跌期权成交量激增 + 正股下跌 → 空头共识
- Put/Call Ratio 极端值（<0.5 或 >2.0）→ 情绪极端

### 1.4 情绪信号（option_sentiment）
**检测规则：**
- 综合期权情绪指标指向单边 → 确认方向
- 情绪与正股价格背离 → 反转预警

### 1.5 综合信号（option_composite）
**检测规则：**
- 多维信号共振 → 高置信度信号
- 信号相互矛盾 → 低置信度，观望

## 维度二：牛熊证异动（仅港股）

**检测规则：**
1. 街货比例突增（日环比 > 20%）→ 散户方向性押注
2. 价格区间集中异动 → 可能触发大规模回收
3. 牛证街货 > 熊证街货 2 倍以上 → 散户偏多，关注杀牛风险
4. 熊证街货 > 牛证街货 2 倍以上 → 散户偏空，关注杀熊风险

**强度评级：**
- **strong**：街货比例极端 + 价格区间集中
- **moderate**：街货比例显著变化但未极端
- **weak**：轻微波动

# Tier 评级调整建议

| 信号条件 | Tier 调整 |
|:---|:---|
| 期权大单 + 量价 + 情绪三维度共振 | **Tier 1 加分**：衍生品层面强烈信号 |
| IV 突升 > 50% + 期权大单 | **Tier 1 警告**：市场预期剧变 |
| 牛熊证街货极端 + 正股接近回收价 | **Tier 1 警告**：衍生品回收风险 |
| 单一维度信号 | **Tier 2 备注**：关注后续确认 |
| 无显著信号 | **不调整** |

> **重要原则**：衍生品信号反映市场参与者的预期和杠杆行为，是辅助信号。只有与基本面/资金面/技术面共振时，才提升 Tier 权重。极端衍生品信号（如 IV 突升）本身可作为风险预警。
