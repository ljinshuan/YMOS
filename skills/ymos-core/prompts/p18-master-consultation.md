# 四大师分析框架

## Lens 1: 巴菲特 — 护城河与所有者收益

> 核心信仰：以合理价格买入优秀公司，长期持有。关键不是便宜，而是"值得持有 10 年"。

### 评估维度

1. **经济护城河**：护城河有多深？是品牌、网络效应、转换成本还是成本优势？护城河 10 年后是否还在？
2. **所有者收益（Owner Earnings）**：净利润 + 折旧摊销 - 维护性资本开支。这不是会计利润，是股东真正能拿走的钱。
3. **资本配置**：管理层如何使用自由现金流？回购时机是否聪明？并购历史回报如何？
4. **管理层诚信**：管理层是否像"所有者"一样思考？还是像"打工者"一样追求规模？
5. **能力圈检验**：这个行业的商业模式你（作为分析师）能理解到什么程度？未来 10 年的盈利图景是否可预见？

### 输出格式

```
### 巴菲特 Lens
- **判断**: bullish / neutral / bearish
- **置信度**: X%
- **Moat Score**: 1-10（护城河综合评分）
- **Owner Earnings Quality**: 1-10（所有者收益质量）
- **核心论点**: 2-3 句话总结巴菲特视角下最看重什么、当前判断是什么
- **关键数据**: 支撑判断的 1-2 个具体数据点
```

---

## Lens 2: 格雷厄姆 — 安全边际与量化筛选

> 核心信仰：投资的本质是"以显著低于内在价值的价格买入"。安全边际是唯一的免费午餐。

### 评估维度

1. **安全边际**：当前价格相对保守估算的内在价值，折让幅度是多少？是否有 30-40% 的安全边际？
2. **量化筛选**：
   - P/E（TTM）是否低于行业均值？
   - P/B 是否低于 1.5 或历史低位？
   - 流动比率是否 > 2？
   - 负债/权益比是否合理？
   - 近 5 年是否持续盈利？
3. **防御型投资者标准**（Graham's Defensive Investor）：
   - 公司规模是否足够大？
   - 财务状况是否足够强？
   - 是否有持续分红记录？
   - 近 10 年是否无亏损年份？
4. **净净值检验**：如果公司今天清算，流动资产 - 全部负债，值多少？当前价格相对这个"地板价"有无折让？

### 输出格式

```
### 格雷厄姆 Lens
- **判断**: bullish / neutral / bearish
- **置信度**: X%
- **Margin of Safety**: X%（安全边际百分比）
- **Graham Score**: 1-10（量化筛选综合评分）
- **核心论点**: 2-3 句话总结格雷厄姆视角下最看重什么、当前判断是什么
- **关键数据**: 支撑判断的 1-2 个具体数据点（P/E、P/B、流动比率等）
```

## Lens 3: 马克斯 — 周期定位与二阶思维

> 核心信仰：投资中最危险的事不是买差公司，而是在错误的时间买好公司。市场周期是可识别但不可预测的。

### 评估维度

1. **周期定位**：当前市场/行业/个股处于周期的什么位置？（六桶分类：冰点 / 低迷 / 回升 / 乐观 / 狂热 / 崩溃）
2. **二阶思维**：
   - 一阶思维："这是一家好公司，买入。"
   - 二阶思维："这是一家好公司，但所有人都知道，所以已经被充分定价了。"
   - 市场共识是什么？共识有多大可能是对的？
3. **风险态度温度计**：当前市场对该标的/行业的风险态度是什么？
   - 过度恐惧（价格已反映最坏预期）→ 机会
   - 过度乐观（价格已反映最好预期）→ 风险
   - 中性 → 无明显机会
4. **逆势检验**：当前的判断是顺大流还是逆共识？如果逆共识，你的"edge"在哪里？
5. **不对称性**：上行空间 vs. 下行风险的比例是多少？是否有"正面我赢很多，反面我输不多"的机会？

### 输出格式

```
### 马克斯 Lens
- **判断**: bullish / neutral / bearish
- **置信度**: X%
- **Cycle Position**: 冰点 / 低迷 / 回升 / 乐观 / 狂热 / 崩溃
- **Contrarian Score**: 1-10（逆向价值评分，越高越有逆向机会）
- **核心论点**: 2-3 句话总结马克斯视角下最看重什么、当前判断是什么
- **二阶洞察**: 市场共识是什么？你看到的被忽视的风险或机会是什么？
```

## Lens 4: 林奇 — 成长分类与 PEG 估值

> 核心信仰：投资你了解的公司。不同类型的公司需要不同的分析框架和持有策略。

### 评估维度

1. **林奇六分类**：这家公司属于哪一类？
   - 慢速增长型（Slow Grower）：年增长 2-3%，稳定分红
   - 稳定增长型（Stalwart）：年增长 10-12%，大型蓝筹
   - 快速增长型（Fast Grower）：年增长 20%+，小而猛
   - 周期型（Cyclical）：盈利随经济周期大幅波动
   - 转机型（Turnaround）：困境反转
   - 资产隐蔽型（Asset Play）：拥有被低估的隐蔽资产
2. **PEG 比率**：P/E ÷ 盈利增长率。PEG < 1 = 可能便宜，PEG > 2 = 可能贵。
3. **故事检验**（Story Test）：用一句话说清"这只股票的故事"——增长从哪来？为什么现在是好时机？如果故事说不清楚，不要买。
4. **可理解性**：公司的业务对普通投资者来说是否可理解？你能否在 2 分钟内向朋友解释清楚它怎么赚钱？
5. **内部人行为**：管理层和内部人最近是在买还是在卖？内部人的行为比他们的话更诚实。

### 输出格式

```
### 林奇 Lens
- **判断**: bullish / neutral / bearish
- **置信度**: X%
- **Lynch Category**: 慢速增长 / 稳定增长 / 快速增长 / 周期型 / 转机型 / 资产隐蔽型
- **PEG Ratio**: X.X（如数据可用）
- **核心论点**: 2-3 句话总结林奇视角下最看重什么、当前判断是什么
- **故事检验**: 一句话说清这只股票的"故事"
```

# Agent 执行指南

1. 读取标的的 `个股基础知识库.md`，获取 P1 七维度分析和 P4 雷达数据作为输入
2. 依次执行 4 个大师 lens，每个 lens 独立分析，不要互相参考
3. 每个 lens 必须输出结构化信号（判断 + 置信度 + 风格专属评分 + 核心论点）
4. 最后输出综合报告，重点标注共识与分歧
5. 将完整的大师会诊报告追加到 `个股基础知识库.md`

## 分析质量标准

- **严格运用框架**：每个 lens 必须从该大师的核心维度出发，不能泛泛而谈
- **具体数据支撑**：每个判断必须有具体数据或事实支撑，不能只有定性描述
- **诚实面对分歧**：如果某个 lens 的数据不足，标注 [数据不足] 而非强行给出判断
- **综合报告聚焦行动**：共识/分歧分析最终要落到"下一步建议"上
