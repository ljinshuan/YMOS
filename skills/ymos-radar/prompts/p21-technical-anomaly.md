# P21: 技术面异动信号分析 (Technical Anomaly)

## 路径上下文（YMOS）
- 根目录：`YMOS/`
- 输入：`ymos fetch-technical-anomaly` 输出的 JSON 数据
- 可选输入：标的 `个股基础知识库.md`（提供基本面上下文）
- 写回：投资雷达报告「技术面信号」区块 + 状态机 P4 更新

## 适用场景

在投资雷达的资金流扫描步骤之后，对持仓和 Watchlist 标的技术面异动数据进行分析。输出技术指标信号、支撑/压力位及 Tier 调整建议。

## 输入数据来源

由 `ymos fetch-technical-anomaly fetch --from-state` 生成，包含每个标的的：
- **K 线形态**：头肩顶/底、双顶/底、三角形突破等
- **技术指标异常**：MACD 金叉/死叉、RSI 超买/超卖、KDJ 钝化、布林带突破等
- **支撑/压力位**：关键技术价位

# Analysis Protocol

按以下维度分析，仅在有明确信号时报告：

## 维度一：K 线形态信号

**信号检测规则：**
1. 反转形态确认（头肩顶/底、双顶/底）→ 趋势反转信号
2. 中继形态（三角形、旗形）突破 → 趋势延续信号
3. 缺口（跳空高开/低开）→ 需结合成交量判断强度

**强度评级：**
- **strong**：多周期共振 + 成交量配合
- **moderate**：单一周期信号 + 部分量能确认
- **weak**：形态不完整或量能不足

## 维度二：技术指标信号

**信号检测规则（按指标）：**

| 指标 | 看多信号 | 看空信号 |
|:---|:---|:---|
| MACD | 金叉（DIF 上穿 DEA） | 死叉（DIF 下穿 DEA） |
| RSI | 从超卖区（<30）回升 | 从超买区（>70）回落 |
| KDJ | 低位金叉 | 高位死叉 / 顶背离 |
| BOLL | 下轨支撑反弹 | 上轨突破后回落 |
| MA | 短期均线上穿长期均线 | 短期均线下穿长期均线 |
| CCI | 从 -100 以下回升 | 从 +100 以上回落 |

**强度评级：**
- **strong**：3+ 个指标同方向共振
- **moderate**：2 个指标同方向
- **weak**：单一指标信号

## 维度三：支撑/压力位

**规则：**
- 标出关键技术价位，评估当前价格与关键位的距离
- 若价格正在测试支撑/压力位，标注突破概率

# Tier 评级调整建议

| 信号条件 | Tier 调整 |
|:---|:---|
| 3+ 指标同向共振 + K 线形态确认 | **Tier 1 加分**：技术面强烈信号 |
| RSI 极端值（<20 或 >85）+ 量价背离 | **Tier 1 警告**：技术面极端状态 |
| 单一指标信号 + 无形态配合 | **Tier 2 备注**：关注后续确认 |
| 无显著信号 | **不调整** |

> **重要原则**：技术面是辅助信号，不构成独立买卖依据。只有当技术面信号与基本面/资金面/事件面信号共振时，才提升 Tier 权重。

## 输入参数
