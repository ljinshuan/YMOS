# P19 — 评论情绪分析 (Comment Sentiment Analysis)

> 结构化分析社区评论的情绪分布，输出多空百分比和关键观点。

## 输入

一组社区评论/帖子数据，包含：
- 评论内容（text）
- 发布时间（timestamp）
- 点赞数/回复数（engagement）
- 作者信息（author，可选）

## 任务

分析这些评论，输出以下结构化结果：

### 1. 情绪分类

将每条评论分类为：
- **Bullish（看多）**：对股票未来表现持乐观态度
- **Bearish（看空）**：对股票未来表现持悲观态度
- **Neutral（中性）**：客观分析、提问、或无明显倾向

计算各类百分比（基于有效评论数，过滤无关/灌水内容）。

### 2. Temperature Score (0-100)

综合情绪强度打分：
- 0 = 极度恐慌（所有人都在抛售/看空）
- 50 = 中性均衡
- 100 = 极度贪婪（所有人都在追涨/看多）

### 3. 关键观点提取

从评论中提取最具代表性的观点（按点赞数/回复数加权）：
- **Top 3 看多观点**：最常被提及的看多理由
- **Top 3 看空观点**：最常被提及的看空理由

### 4. 一句话结论

综合情绪分布和关键观点，用一句话概括当前市场情绪状态。

## 输出格式

```markdown
## 情绪分布
- 看多: XX%（XX 条有效评论）
- 看空: XX%（XX 条）
- 中性: XX%（XX 条）
- 已过滤无关评论: XX 条
- Temperature Score: XX/100

## 关键看多观点
1. "[引用原话或概括]" — 被提及 XX 次
2. "[引用原话或概括]" — 被提及 XX 次
3. "[引用原话或概括]" — 被提及 XX 次

## 关键看空观点
1. "[引用原话或概括]" — 被提及 XX 次
2. "[引用原话或概括]" — 被提及 XX 次
3. "[引用原话或概括]" — 被提及 XX 次

## 一句话结论
[综合情绪总结]

## 预警（如有）
⚠️ 市场情绪极度[乐观/悲观]（看[多/空]占比 XX%），注意[反向风险/逆向机会]
```

## 注意事项

- 评论数据来自社区，不代表专业投资建议
- 情绪分析基于文本内容，可能存在采样偏差
- Temperature Score 仅反映当前评论样本的情绪倾向，不预测未来走势
- 极端情绪（看多 > 80% 或看空 > 70%）通常是反向指标，而非跟随信号
