# P19 — 评论情绪分析 (Comment Sentiment Analysis)

> 结构化分析社区评论的情绪分布，输出多空百分比和关键观点。

## 输入

一组社区评论/帖子数据，包含：
- 评论内容（text）
- 发布时间（timestamp）
- 点赞数/回复数（engagement）
- 作者信息（author，可选）

## 任务

分析这些评论，输出以下结构化结果：

### 1. 情绪分类

将每条评论分类为：
- **Bullish（看多）**：对股票未来表现持乐观态度
- **Bearish（看空）**：对股票未来表现持悲观态度
- **Neutral（中性）**：客观分析、提问、或无明显倾向

计算各类百分比（基于有效评论数，过滤无关/灌水内容）。

### 2. Temperature Score (0-100)

综合情绪强度打分：
- 0 = 极度恐慌（所有人都在抛售/看空）
- 50 = 中性均衡
- 100 = 极度贪婪（所有人都在追涨/看多）

### 3. 关键观点提取

从评论中提取最具代表性的观点（按点赞数/回复数加权）：
- **Top 3 看多观点**：最常被提及的看多理由
- **Top 3 看空观点**：最常被提及的看空理由

### 4. 一句话结论

综合情绪分布和关键观点，用一句话概括当前市场情绪状态。

## 输出格式

```markdown
## 情绪分布
- 看多: XX%（XX 条有效评论）
- 看空: XX%（XX 条）
- 中性: XX%（XX 条）
- 已过滤无关评论: XX 条
- Temperature Score: XX/100

## 关键看多观点
1. "[引用原话或概括]" — 被提及 XX 次
2. "[引用原话或概括]" — 被提及 XX 次
3. "[引用原话或概括]" — 被提及 XX 次

## 关键看空观点
1. "[引用原话或概括]" — 被提及 XX 次
2. "[引用原话或概括]" — 被提及 XX 次
3. "[引用原话或概括]" — 被提及 XX 次

## 一句话结论
[综合情绪总结]

## 预警（如有）
⚠️ 市场情绪极度[乐观/悲观]（看[多/空]占比 XX%），注意[反向风险/逆向机会]
```

### Group 聚合模式（多标的）

当输入包含多个标的的评论数据时，在完成各标的分析后，额外输出 Group 聚合层：

```markdown
## Group 情绪聚合（多标的）
- 分析标的数: X
- 总有效评论数: XX
- 聚合情绪: [bullish / bearish / neutral / mixed]
- 聚合看多占比: XX%
- 聚合看空占比: XX%
- 聚合中性占比: XX%

### 驱动标的
| 标的 | 情绪偏差 | 说明 |
| [TICKER] | 看多偏离 +XX% | 该标的情绪显著偏离组均值 |
```

**聚合规则**：
1. 按各标的评论数加权计算聚合百分比
2. 情绪标签判定优先级：明确优势（多空差 ≥ 15%）> 混合（多空均 ≥ 25% 且差 < 15%）> 弱优势 > 中性
3. **mixed 标签**：当看多和看空均 ≥ 25% 且差距 < 15% 时使用，表示市场分歧显著
4. 驱动标的：情绪偏离组均值 > 15% 的标的

## 注意事项

- 评论数据来自社区，不代表专业投资建议
- 情绪分析基于文本内容，可能存在采样偏差
- Temperature Score 仅反映当前评论样本的情绪倾向，不预测未来走势
- 极端情绪（看多 > 80% 或看空 > 70%）通常是反向指标，而非跟随信号
