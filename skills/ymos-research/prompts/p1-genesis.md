# 第一层：投资机会分类

## 分类判断流程

当分析一家公司时，按以下步骤判断其所属类别（可多选）：

1. **是否具备深厚护城河、稳定增长、优秀管理层和资本配置记录？** → 考虑「高质量复利型」
2. **收入是否在快速增长且有清晰的增长驱动力？** → 考虑「成长机会型」
3. **是否存在市场明显误判或公司正在执行重大改善计划？** → 考虑「估值重塑型」
4. **当前是否处于周期底部、估值和盈利双低？** → 考虑「周期复苏型」

## 四大类别定义

### 类别一：高质量复利型（Quality Compounder）

**核心定义**：具备深厚护城河、稳定增长能力、优秀管理层的长期持有型标的。目标是通过 EPS 稳定增长 + 股东回报（分红/回购）实现年化 10%+ 的总回报。

**选股标准**：护城河深、议价能力强；经常性收入（Recurring Revenue）占比高；优秀的内部管理和企业文化；优秀的资本配置（Capital Allocation）历史记录；年化总回报预期 ≥ 10%。

**分类后的分析侧重**：第二层分析中，重点关注「治理结构」「商业壁垒」「财务健康」三个维度。核心关注长期逻辑是否稳固、护城河是否有被侵蚀的迹象。

**典型持有策略**：SOP 1（长期持有 3–5 年+）。关注重心在长期逻辑的稳固性，短期波动容忍度高。

**核心风险**：长期投资逻辑破裂，导致持续大幅 de-rating。

**参考案例**：TSMC, APPL，Coca-Cola（10Y 年化回报约 10% = 6% EPS CAGR + 3% Div/Buyback Yield，P/E 21–25x）

### 类别二：成长机会型（Growth）

**核心定义**：收入快速增长且能最终兑现盈利的高成长标的。增长驱动力来自 TAM 扩张、技术迭代或市占率提升。

**选股标准**：收入快速增长（TAM 扩张、新产品、区域拓展、技术迭代、市占率提升）；能兑现盈利增长——具备护城河抵御竞争（生产资料优势、技术壁垒、用户粘性）；供给存在瓶颈时量价齐升、利润弹性大；固定成本占比高，收入增长带来 Operating Leverage。

**分类后的分析侧重**：第二层分析中，重点关注「成长抓手」「商业壁垒」「博弈维度」三个维度。核心关注增长持续性、竞争格局、以及稳态利润率/ROIC。

**典型持有策略**：SOP 2（机会性持有 6–18 个月），增长逻辑兑现后（高成长的visibility较强）可升级为 SOP 1。

**核心风险**：高成长逻辑破裂；无法兑现 Profitable Growth（高成长吸引更多竞争，若供给无限制则可能出现价格战）。

**参考案例**：Nvidia, Uber, GE Vernova

### 类别三：估值重塑型（Turn-around / Re-rating）

**核心定义**：市场对公司存在误判或公司正在执行业务改善计划，存在估值修复的机会。

**选股标准（两种子类型）**：
- **子类型 A — 错杀（Market Mispricing）**：市场对公司长期逻辑（增长前景、竞争威胁、破产风险等）过度担忧，估值已过度反映悲观预期。
- **子类型 B — 业务改善（Operational Turnaround）**：现有业务管理不善，但公司已提出明确改善计划，或计划通过分拆/重组释放隐藏价值。

**分类后的分析侧重**：第二层分析中，重点关注「博弈维度」「治理结构」「商业壁垒」三个维度。核心关注市场的误判点在哪里、业务改善的可行性、以及业务本身的质量上限。

**典型持有策略**：SOP 2（机会性持有），催化剂兑现窗口明确。需要持续验证市场误判是否在修正。

**核心风险**：估值重塑逻辑迟迟不兑现；市场担忧被证实，估值继续下行。此类投资更需要独立思考和逆向判断。

**参考案例**：Siemens Energy, GE, Nidec

### 类别四：周期复苏型（Cyclical Recovery）

**核心定义**：处于周期底部的强周期股，估值和盈利双低，等待周期上行带来的量价齐升和估值修复。

**选股标准**：属于强周期行业；当前处于周期底部（估值低、盈利低）；存在周期复苏的合理预期。

**分类后的分析侧重**：第二层分析中，重点关注「周期维度」「博弈维度」两个维度。核心关注当前周期位置、反转信号、以及最关键的——区分周期性 vs. 结构性下滑。

**典型持有策略**：SOP 2（机会性持有），左侧布局、右侧加仓、周期高点前退出。时机判断是核心。

**核心风险**：下行周期持续或复苏时点不断推后；最大风险是最初认为的周期性下滑最终被证明是结构性原因导致的永久性衰退。

**参考案例**： Rockwell Automation, Frontline

# 第二层：七维度深度分析

> Agent 在完成第一层分类后，对每家公司进行以下七个维度的深度分析。根据分类结果调整各维度的分析深度——被标注为「重点关注」的维度应做更详尽的研究。

## 维度一：治理结构 (The Soul) — 掌舵人与团队

> **分析目标**：评估公司的灵魂人物和管理团队是否值得信赖、是否有能力带领公司穿越周期。
> **重点适用类别**：Quality Compounder（管理层决定长期价值创造）、Turn-around（管理层决定改善计划能否执行）

### 1.1 Missionary vs. Mercenary

创始人/CEO 是技术信仰者（如 Musk, Hassabis）还是职业经理人？具体评估：
- 在公司"至暗时刻"或"技术分叉口"的具体表现是什么？是否展现出长期信念和战略定力？
- 对行业和技术的理解深度如何？是否在关键决策中展现出远见？

### 1.2 核心团队稳定性

- 核心管理层和技术骨干的背景（从业经验、行业出身）
- 关键人员的流失率。近 2–3 年是否有核心高管/技术骨干离职？离职原因？
- 团队的互补性如何？是否存在关键人物风险（Key Person Risk）？

### 1.3 管理风格与执行力

- 管理团队的能力、激励机制、企业文化是否支持长期价值创造？
- **Guidance 执行力**：过去几个季度是否 beat guidance？管理层倾向于 under-promise 还是 over-promise？
- 资本配置（Capital Allocation）决策质量：历史上的 M&A、分红、回购决策回报如何？

### 1.4 股权结构

- 机构持股比例及变化趋势
- 近期内部人交易（Insider Trading）情况：管理层是在买入还是卖出？
- 是否存在控股股东或投票权集中的问题？

---

## 维度二：商业壁垒 (The Moat) — 商业模式与护城河

> **分析目标**：评估公司赚钱的核心逻辑和竞争优势的可持续性。
> **重点适用类别**：Quality Compounder（护城河是长期复利的基础）、Growth（护城河决定能否兑现 Profitable Growth）

### 2.1 商业模式简述

用通俗语言提炼其核心盈利逻辑——靠什么赚钱？价值链中处于什么位置？收入结构是怎样的（产品/服务/订阅/交易佣金等）？

### 2.2 灵魂拷问：10 年后它为什么还能屹立不倒？

回答这个问题需要判断：公司的核心竞争优势是在加强还是减弱？行业格局的演变方向是否对公司有利？

### 2.3 竞争格局

- 市场格局是否集中？是否稳定？CR3/CR5 是多少？
- 公司在行业中排第几？
- 有没有新进入者？新进入者的威胁有多大？
- 行业是否存在整合趋势？

### 2.4 护城河分析

逐项评估以下护城河来源（标注强/中/弱/无）：

| 护城河类型 | 评估 | 具体表现 |
|-----------|------|---------|
| 网络效应 | | 用户/供应商越多，价值越大？ |
| 转换成本 / 用户粘性 | | 客户迁移的成本和摩擦有多大？ |
| 技术壁垒 / 知识产权 | | 专利、技术积累、know-how |
| 生产资料 / 产能优势 | | 稀缺资源、产能瓶颈、工人/专家壁垒 |
| 品牌 / 认证壁垒 | | 行业认证、品牌忠诚度、准入门槛 |
| 规模经济 / 成本优势 | | 规模带来的成本领先 |
| 数据壁垒 | | 独有数据积累形成的优势 |

### 2.5 Pricing Power（定价权）

- 成本上升时能否提价？利润率会被挤压吗？
- 过去 3–5 年价格变化趋势如何？
- 定价权的来源是什么（供给紧张、品牌、转换成本、合同结构等）？

### 2.6 长期叙事风险监控

是否存在可能颠覆公司护城河或长期投资逻辑的风险因素？例如：
- 技术替代风险（如 AI 对软件公司的颠覆）
- 监管/政策变化
- 行业结构性变化（如新能源对传统能源的替代）
- 新商业模式的冲击

## 维度三：成长抓手 (The Driver) — 增长来源

> **分析目标**：拆解增长的来源、持续性和可见度。
> **重点适用类别**：Growth（增长是核心投资逻辑）、Quality Compounder（增长稳定性是复利的前提）

### 3.1 增长来源拆解

- **核心问题**：增长从哪里来？
- 过去 3–5 年的增长主要靠什么？（TAM 增长？市占率提升？量的增长？提价？并购？）
- 未来 3–5 年的增长引擎是什么？增长来源是否在发生切换？

### 3.2 成长持续性与 Visibility

- 高成长能看几年？增长的 visibility 如何？
- 需求会不会突然变化？有没有技术迭代风险？
- 是否有积压订单（backlog）、长期合同等提供收入可见性？

### 3.3 TAM 增长拆分

- 未来 3 年 TAM 增长预期（乐观/中性/悲观情景）
- 自上而下拆分：终端需求是什么？驱动力是什么？
- 技术/产品渗透率提升空间有多大？

### 3.4 市占率提升潜力

- 靠什么提升市占率？技术迭代？新产品？区域拓展？成本优势？
- 市占率提升是可持续的还是一次性的？

### 3.5 利润率扩张逻辑

- 收入增长能否带来利润率提升？还是靠牺牲盈利换增长？
- 如果供给有瓶颈、供不应求，是否体现出量价齐升？
- 如果固定成本较高，是否有 Operating Leverage？
- 稳态下的利润率 / ROIC 预期是多少？（取决于商业模式和竞争格局）

## 维度四：周期维度 (The Cycle) — 市场与周期位置

> **分析目标**：判断公司所处的周期阶段，区分周期性与结构性因素。
> **重点适用类别**：Cyclical Recovery（这是核心分析维度）；对所有类别都应有基本的周期意识

### 4.1 历史周期分析

- 过往周期的驱动因素是什么？（需求驱动？供给冲击？政策变化？）
- 历史上量、价、利润率的弹性如何？上行周期和下行周期各自的幅度和持续时间？
- 例如：供给冲击导致能源价格飙升；需求暴涨叠加供给不足导致集运价格暴涨

### 4.2 当前周期位置判断

从四个维度判断当前所处阶段：

| 维度 | 当前状态 | 相对历史位置 |
|------|---------|-------------|
| 量（Volume） | | 低谷/复苏/高峰/下行 |
| 价（Price） | | 低谷/复苏/高峰/下行 |
| 利润率（Margin） | | 低谷/复苏/高峰/下行 |
| 估值（Valuation） | | 低谷/复苏/高峰/下行 |

### 4.3 周期反转信号

- 反转信号是什么？有哪些先行指标（Leading Indicators）可以追踪？
- 当前处于左侧还是右侧？
- 是需求反转还是供给反转？
- 有什么具体的 Catalyst？
- 赢率和赔率分析：向上空间 vs. 向下风险的不对称性

### 4.4 区分周期性 vs. 结构性（最关键）

**这是最关键的分析**——当前的变化（无论是上行还是下行）究竟是周期性的还是结构性的？
- 如果是下行：最大风险是最初认为的周期性下滑最终被证明是结构性衰退
- 如果是上行：当前的强劲表现是周期高点还是新结构性趋势的起点？
- 支持周期性判断的证据是什么？支持结构性判断的证据是什么？

## 维度五：财务健康 (The Fuel) — 财务与核心指标

> **分析目标**：从财务角度评估公司的盈利质量、现金流质量和资本效率。
> **重点适用类别**：Quality Compounder（FCF 和 ROIC 是核心）、Growth（关注盈利兑现路径）

### 5.1 现金流质量

- 经营性现金流（OCF）与自由现金流（FCF）的长期趋势
- FCF Conversion（FCF / 净利润）的质量如何？
- 资本开支（CapEx）的用途——维护性 vs. 扩张性 CapEx 的比例？

### 5.2 资本效率

- ROIC（投入资本回报率）趋势及与 WACC 的对比
- 研发投入产出比（如适用）
- 资本配置历史记录：M&A 回报、分红/回购策略

### 5.3 北极星指标（North Star Metric）

除了通用财务数据，该行业的第一性原理指标是什么？Agent 应根据行业特性选择：

| 行业       | 北极星指标示例                                    |
| -------- | ------------------------------------------ |
| SaaS     | NDR（净收入留存率）、ARR、NRR、CAC Payback、Rule of 40 |
| 电动车      | 单车毛利、交付量、产能利用率                             |
| AI / 云计算 | 推理成本变化、GPU 利用率、ARR 增速                      |
| 半导体      | 产能利用率、ASP 趋势、设计胜出数                         |
| 航空材料     | 每磅售价/利润、产能利用率、交货周期                         |
| 零售/消费    | 同店增长、客单价ARPU、复购率                           |
| 金融       | ROE、NIM、信贷质量、资本充足率                         |
| 电商       | GMV、Take Rate、DAU/MAU                      |
| 广告       | DAU/MAU、ARPU、广告加载率load factor              |
| 制造业      | 产能利用率、库存、渠道库存                              |

### 5.4 宏观敏感度

- 对利率变化的敏感度（融资成本、折现率影响）
- 对政策和监管变化的暴露程度
- 对地缘政治风险的暴露程度（供应链、市场准入、关税等）
- 对汇率波动的敏感度

## 维度六：博弈维度 (The Alpha) — 共识与分歧

> **分析目标**：识别市场共识与潜在预期差，评估当前定价的合理性。
> **重点适用类别**：Turn-around（预期差是超额回报的核心来源）、Cyclical Recovery（时机判断和赔率分析）

### 6.1 Bull Case（看多逻辑）

市场主流的买入理由是什么？支撑这些理由的核心假设是什么？

### 6.2 Bear Case（看空逻辑）

市场主要的担忧与风险点是什么？看空者的核心论据是什么？

### 6.3 预期差分析

- 当前估值主要计入了哪种预期（乐观/中性/悲观）？
- 市场可能忽略或低估了什么？
- 市场可能高估或过度乐观了什么？
- 如果存在预期差，催化剂（Catalyst）是什么？什么事件可能让市场重新定价？

## 维度七：估值与回报拆解 (The Price) — 估值和市场预期

> **分析目标**：量化当前市场定价隐含的预期，拆解未来回报的来源（利润增长 vs. 估值变化），识别尚未被 priced in 的机会与风险。
> **重点适用类别**：所有类别均需完成此维度。Turn-around 和 Cyclical Recovery 尤其关注估值弹性；Quality Compounder 和 Growth 关注利润增长对回报的贡献。

### 7.1 Consensus 预期与预期差

- 未来 3 年市场一致预期：Sales CAGR、EPS CAGR、利润率变化趋势
- 一致预期是否合理？哪些假设可能过于乐观或过于保守？
- **尚未被 Priced In 的机会**：市场可能低估了什么？（例如：新产能投产、市占率拐点、定价权提升、政策红利）
- **尚未被 Priced In 的风险**：市场可能忽视了什么？（例如：竞争加剧、周期见顶、客户集中度、监管变化）

### 7.2 公司指引/目标

- 当年收入/利润率指引
- 中期指引（什么时候给出的，覆盖什么时间段）

### 7.3 估值水平与历史对比

- 当前 P/E（NTM）以及基于 2 年后、3 年后 Consensus EPS 的隐含 P/E
- 与历史估值区间对比：当前处于什么分位？
- 与可比公司（Peers）对比：溢价/折价是否合理？溢价/折价的原因是什么？
- 估值方法适用性：P/E 是否是最合适的估值方法？是否需要辅以 EV/EBITDA、P/FCF、DCF 等？

### 7.4 回报来源拆解

将未来 1–3 年的潜在回报拆解为三个来源：

```
预期年化总回报 = EPS 增长贡献 + 估值变化贡献 + 股东回报贡献（分红 + 回购）
```

|---------|-----------|-----------|-----------|
| EPS 增长（CAGR） | | | |
| 估值变化（P/E 变化） | | | |
| 分红 + 回购收益率 | | | |
| **年化总回报** | | | |

- **Upside 主要来自利润还是估值？** 如果主要靠估值扩张，风险更高（估值是市场情绪，不可控）；如果主要靠利润增长，确定性更强。
- **Downside 主要来自利润还是估值？** 利润下滑 + 估值压缩的"双杀"情景有多大概率？触发条件是什么？
