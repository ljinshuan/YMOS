## ADDED Requirements

### Requirement: P8 辩论模式 prompt 结构
P8 Macro Filter prompt MUST 包含辩论模式章节，在原有三维度评级之后增加多学派视角分析。

#### Scenario: 辩论模式自动触发
- **WHEN** Agent 执行 P8 Macro Filter
- **THEN** 在完成贴现率敏感度、业务传导机制、避险属性三维度评级后，自动执行辩论模式

### Requirement: 三学派视角定义
P8 辩论模式 MUST 包含以下 3 个经济学派视角：
- 凯恩斯主义：从需求侧、财政乘数、反周期政策角度分析
- 奥地利学派：从供给侧、货币信用、市场自发秩序角度分析
- 重商主义：从国家战略、贸易顺差、关键产业保护角度分析

#### Scenario: 三学派独立输出
- **WHEN** 辩论模式执行
- **THEN** 输出 MUST 包含 3 个独立的学派分析 section，每个 section 从该学派的理论框架出发分析同一宏观问题

### Requirement: Steelman 反驳机制
P8 辩论模式 MUST 输出"最强反对论点"——即与原始评级最有力的对立观点。

#### Scenario: 存在明确反对论点
- **WHEN** 原始评级为"有利"，但奥地利学派提出了有说服力的反面论证
- **THEN** 输出 MUST 包含"最强反对论点"section，完整呈现该反对论点的逻辑链条

### Requirement: 辩结论输出
P8 辩论模式 MUST 在辩论后输出修正或确认的最终结论，包含：
- 最强反对论点
- 该论点的处理：被采纳（导致评级调整）或被否决（附理由）
- 最终评级（可能与原始评级相同或不同）

#### Scenario: 辩论改变原始评级
- **WHEN** 原始评级为"中性"，但辩论发现被忽视的系统性风险
- **THEN** 最终评级调整为"高风险"，并在辩结论中说明调整理由

#### Scenario: 辩论确认原始评级
- **WHEN** 三个学派的分析均未提出有力反对论点
- **THEN** 最终评级与原始评级一致，辩结论说明反对论点被否决的理由
