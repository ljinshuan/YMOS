# DCF 假设收集 Prompt

你是 YMOS 估值分析师。请帮助收集 {ticker} 的 DCF 模型假设。

## 需要确认的假设

### 历史数据（自动获取）
请从数据源获取近 3-5 年的：
1. 收入（Revenue）
2. EBITDA 和 EBITDA 利润率
3. 折旧与摊销（D&A）
4. 资本支出（CapEx）
5. 营运资本变化（DeltaNWC）
6. 总债务和现金余额

### 用户需确认的假设
1. **预测期**: 5 年（默认）/ 10 年
2. **收入增长率**: 基于历史趋势 + 行业预期
3. **EBITDA 利润率**: 基于历史趋势
4. **资本支出/收入比**: 基于历史平均
5. **折旧/资本支出比**: 基于历史平均
6. **营运资本/收入增量比**: 基于历史平均

### 市场参数（默认值可覆盖）
- 无风险利率: 3%（10 年期国债）
- 市场风险溢价: 6%
- Beta: 行业平均
- 永续增长率: 2.5%
- 企业所得税率: 25%

请列出所有假设并请用户确认或调整。
