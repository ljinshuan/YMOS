# P20: 资金异动信号分析 (Capital Flow Anomaly)

## 路径上下文（YMOS）
- 根目录：`YMOS/`
- 输入：`ymos fetch-capital-flow` 输出的 JSON 数据
- 可选输入：标的 `个股基础知识库.md`（提供基本面上下文）
- 写回：投资雷达报告「资金流扫描」区块 + 状态机 P4 更新

## 适用场景

在投资雷达的价格扫描步骤之后，对持仓和 Watchlist 标的资金流数据进行异动分析。输出异动信号、强度评级和 Tier 调整建议。

## 输入数据来源

由 `ymos fetch-capital-flow --from-state` 生成，包含每个标的的：
- **资金分布**：大单/特大单/中单/小单净流入流出
- **买卖经纪商**：Top 5 买入/卖出经纪商席位
- **资金流向趋势**：近 N 日主力资金流向变化
- **卖空数据**：卖空数量、卖空比例

# Analysis Protocol

按以下三个维度逐项分析，仅在有明确异动时报告：

## 维度一：资金分布与经纪商活动

**信号检测规则：**
1. 大单/特大单方向与小单方向持续相反（连续 ≥ 2 日）→ 主力与散户分歧
2. Top 5 买入经纪商中出现「沪深港通」「中投」等机构席位 → 机构进场信号
3. Top 5 卖出经纪商集中出现外资席位 → 机构离场信号
4. 单日特大单净流出超过流通市值 1% → **强信号**

**强度评级：**
- **strong**：特大单连续 3 日以上单边净流入/流出 + 机构席位验证
- **moderate**：大单连续 2 日单边 + 经纪商方向一致
- **weak**：单日异动，无连续性

## 维度二：资金流向趋势

**信号检测规则：**
1. 主力资金连续 3 日净流入 → 看多信号
2. 主力资金连续 3 日净流出 → 看空信号
3. 当日净流入/流出金额超过前 3 日均值的 100%（翻倍）→ 加速信号
4. 资金流向与价格走势背离（价涨资金出 / 价跌资金进）→ 趋势反转预警

**强度评级：**
- **strong**：连续 ≥ 5 日同向 + 加速
- **moderate**：连续 3-4 日同向
- **weak**：连续 2 日同向或加速但无连续性

## 维度三：卖空情况

**信号检测规则：**
1. 卖空数量和卖空比例同时异动 → 强看空信号
2. 卖空比例突增至近 1 月均值 2 倍以上 → 做空压力骤增
3. 卖空数量持续下降 + 价格企稳 → 空头回补信号

**强度评级：**
- **strong**：数量和比例同时异动 + 高于月均值 2 倍
- **moderate**：单一指标异动且高于月均值 1.5 倍
- **weak**：指标轻微波动，无明显趋势

# Tier 评级调整建议

根据检测到的信号，对标的的 Tier 评级提出调整建议：

| 信号条件 | Tier 调整 |
|:---|:---|
| 主力连续 3 日净流入 + 机构席位买入 | **Tier 1 加分**：资金面确认了基本面/技术面信号 |
| 单日大单净流出 > 流通市值 1% | **Tier 1 警告**：资金面出现严重预警 |
| 卖空量价同时异动 | **Tier 1 警告**：空头压力骤增 |
| 资金流向与价格背离（价跌资金进） | **Tier 2 备注**：关注趋势反转可能 |
| 无显著异动 | **不调整** |

> **重要原则**：资金流是辅助信号，不构成独立买卖依据。只有当资金流信号与基本面/技术面/事件面信号方向一致时，才提升 Tier 权重。

## 输入参数

| 参数 | 说明 | 必填 |
|-----|------|-----|
| `ticker` | 股票代码 | 是 |
| `capital_flow_data` | `ymos fetch-capital-flow` 输出的 JSON | 是 |
| `time_range` | 时间范围（天），默认 7 | 否 |
| `stock_knowledge_base` | 个股基础知识库内容 | 否 |

## 阈值速查

| 检测项 | 阈值 | 信号方向 |
|:---|:---|:---|
| 主力连续净流入 | ≥ 3 日 | 看多 |
| 主力连续净流出 | ≥ 3 日 | 看空 |
| 单日大单净流出 | > 流通市值 1% | 强看空 |
| 资金流向加速 | 超前 3 日均值 100% | 趋势强化 |
| 价量背离 | 价格与资金方向相反 ≥ 2 日 | 趋势反转预警 |
| 卖空量价同时异动 | 数量 + 比例均异动 | 强看空 |
| 卖空比例突增 | > 月均值 2 倍 | 看空压力 |
| 卖空持续下降 + 价稳 | 连续下降 | 空头回补（看多） |
