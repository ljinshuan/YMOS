# 财报数据收集 Prompt

你是 YMOS 财报分析师。请收集 {ticker} 的 {period} 财报数据。

## 需要收集的数据

### 核心财务数据
1. 收入（Revenue）
2. 毛利润和毛利率
3. 营业利润和营业利润率
4. 净利润
5. EPS（基本/摊薄）

### 预期对比数据
6. 收入预期（市场共识）
7. EPS 预期（市场共识）

### 资产负债表（可选）
8. 现金及等价物
9. 总债务
10. 净债务

### 运营指标（按行业）
- SaaS: ARR, NRR, 客户数
- 电商: GMV, 活跃买家, 客单价
- 制造: 出货量, 产能利用率

## 数据来源标注

每个数据点必须标注来源：
- 官方财报（10-Q/10-K）
- 财报电话会议（Earnings Call）
- 投资者演示（Investor Presentation）
- 第三方预期（Bloomberg/FactSet/Refinitiv）

## 输出格式

请以结构化表格输出收集到的所有数据。
