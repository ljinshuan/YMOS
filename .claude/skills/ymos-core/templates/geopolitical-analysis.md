# 地缘政治分析模板

## 触发条件

当日新闻中涉及以下任意关键词域时激活本模板，对 P13 的"市场风向"维度进行地缘政治信号增强：

| 关键词域 | 关键词 |
|---------|--------|
| 军事冲突 | 军事冲突、战争、军事演习、导弹试射、空袭、封锁、开火、交火 |
| 经制裁衡 | 制裁、出口管制、技术封锁、实体清单、禁运 |
| 贸易摩擦 | 关税、贸易战、加征关税、对等关税、反制关税 |
| 领土争端 | 领土争端、主权声索、划界、防空识别区 |
| 关键地区 | 中东、台海、南海、乌克兰、朝鲜半岛、红海、霍尔木兹海峡 |
| 关键资源 | 石油、稀土、芯片、粮食、天然气、铀、锂、铜 |

若新闻中未命中上述关键词域，跳过本模板，不输出地缘政治信号。

## 分析框架（三步法）

### 第一步：地理约束
分析事件的地理基础：
- 资源分布（该地区控制了什么关键资源？占比多少？）
- 海峡通道（是否影响关键航运通道？如霍尔木兹海峡、马六甲海峡、红海）
- 邻国关系（牵涉哪些国家的直接利益？联盟格局如何？）

### 第二步：历史模式
回溯过去 50 年内的 1-2 个历史类比事件，说明：
- 当时的背景与触发点
- 市场的实际反应路径（而非预设"市场一定会怎样"）
- 当前事件与历史的异同点

### 第三步：传导路径
追踪事件对市场的具体传导链，必须落脚到行业和具体标的：

```
事件 → 商品/供应链影响 → 行业影响 → 持仓/观察列表标的
```

传导路径示例：
- 中东冲突升级 → 油价上涨 → 航空/运输成本上升（利空航空股如 DAL、UAL）、能源股受益（如 XOM、CVX）
- 台海紧张 → 芯片供应链风险 → 半导体设备（如 ASML、AMAT）承压、国产替代受益（如 688008.SS）
- 稀土出口管制 → 稀土价格上行 → 新能源电机成本上升（如 NIO）、稀土矿企受益

## 输出格式

当触发条件满足时，在 P13 报告的"市场风向"维度末尾追加以下 section：

```markdown
### 地缘政治信号
- **事件**: [一句话概括事件]
- **地理约束**: [地理基础分析：资源分布/海峡通道/邻国关系]
- **历史回声**: [1-2 个历史类比，简述当时走势与异同]
- **传导路径**: [事件] → [商品/供应链] → [行业] → [持仓/观察列表标的]
- **盲点提示**: [容易被忽略的风险因素，如二级制裁、替代供应源、时间差等]
```
